Even a stopped clock

Sweetclocks.party helps you keep time on your friends, relations, or collegues around the world. I wrote it after getting tired of guessing what time zones my friends and family were in. Sweetclocks lets you search for locations, predict times, and remembers what clocks you had opened last. Thanks for checking it out!

Sweetclocks is new and still has some bugs. If you'd like to report a bug, please send mail to whoops@sweetclocks.party

You can check out the code for sweetclocks on github

How to use Sweetclocks

Add Clock To add a new clock, click Add Button
Remove Clock To remove a clock, click Remove Button
Time Travel To predict a time click on the clock or on the Time Travel Icon Type the time and press Return or click the Start Time Travel Icon To return to the current time, press Esc or click on the Cancel Time Travel Icon
Search Time Zones To search for a time zone type in the search field above the clock time. Select your time zone from the results list, and the clock should show the time for that region. Keep in mind countries may have more than one time zone.
Rename Time Zones To rename a time zone, edit the name and press Enter without selecting a time zone from the list.
Change UTC/GMT To manually change the UTC/GMT of a zone, click the UTC time offset and select a UTC from the dropdown
Daylight Savings Time If a time zone is currently within Daylight Savings Time (DST), the DST Icon will light. Please note that currently Sweetclocks only checks for DST when you first search for a time zone. If you pass into or out of DST after searching it won't update. To manually enable/disable DST simply click the DST Icon
Save Clock Block me should save to your browsers Local Storage automatically. If you aren't seeing the same settings when you come back to Clock Block using the same Browser, check to see if you have Local Storage disabled.
