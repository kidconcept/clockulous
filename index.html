Even a stopped Clock

Clockblock.me is a small javascript application that helps you keep time on your friends, relations, or collegues around the world. It's open source, and the code is free to use for non comercial, with attribution projects. Please check out the code on github clockulous. Copyright Creative Commons.

How to Use Clockblock

Feature Description
Add Clock To add a new clock, click the Add Button.
Remove Clock To remove a clock, click the Remove Button.
Time Travel To check what time it will be click on a clock time, or on a time travel icon . Then type the time and press Return or click the submit time-travel icon To return to the current time, press Esc or click on the cancel time-travel icon.
Search Time Zones To search for a time zone type in the search field above the clock time. Select your time zone from the results list, and the clock should show the time for that region. Keep in mind countries may have more than one time zone.
Rename Time Zones To rename a time zone, type in the search field above the clock time and press Enter. Don't select a time zone from the drop down.
Daylight Savings Time If a time zone is currently within Daylight Savings Time (DST), the icon will light. Please note that currently the app only checks for DST when you first search for a time zone. If you pass into or out of DST after searching it won't update. To manually enable DST click the
